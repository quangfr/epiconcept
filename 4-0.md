# Qualification d'un Bug
Distinguer la criticité d'un dysfonctionnement selon son impact réel sur l'activité des utilisateurs.

<!--
type: html
-->

<div class="grid grid-cols-1 md:grid-cols-2 gap-4 text-left w-full max-w-5xl mx-auto p-1" style="font-family: sans-serif;">
  <!-- Bloquant vs Non Bloquant -->
  <div class="bg-white/90 border border-slate-200/60 p-4 rounded-xl shadow-xs">
    <div class="flex items-center gap-2 mb-3">
      <span class="px-2 py-0.5 rounded bg-red-100 text-red-700 font-extrabold text-[9px] uppercase tracking-wider">Bloquant</span>
      <span class="text-slate-400 text-xs">vs</span>
      <span class="px-2 py-0.5 rounded bg-blue-100 text-blue-700 font-extrabold text-[9px] uppercase tracking-wider">Non Bloquant</span>
    </div>
    <div class="space-y-3">
      <div>
        <h4 class="font-extrabold text-xs text-slate-800 flex items-center gap-1.5"><i data-lucide="ban" class="w-3.5 h-3.5 text-red-500"></i> Bug Bloquant</h4>
        <p class="text-slate-500 text-[9px] mt-0.5">L'utilisateur est totalement arrêté. Aucune solution de contournement (workaround) fonctionnelle n'existe pour poursuivre la tâche.</p>
      </div>
      <div class="border-t border-slate-100 pt-2">
        <h4 class="font-extrabold text-xs text-slate-800 flex items-center gap-1.5"><i data-lucide="check" class="w-3.5 h-3.5 text-blue-500"></i> Bug Non Bloquant</h4>
        <p class="text-slate-500 text-[9px] mt-0.5">L'utilisateur rencontre une gêne ou un dysfonctionnement, mais il existe un parcours alternatif ou une saisie intermédiaire pour accomplir son travail.</p>
      </div>
    </div>
  </div>

  <!-- Majeur vs Mineur -->
  <div class="bg-white/90 border border-slate-200/60 p-4 rounded-xl shadow-xs">
    <div class="flex items-center gap-2 mb-3">
      <span class="px-2 py-0.5 rounded bg-amber-100 text-amber-700 font-extrabold text-[9px] uppercase tracking-wider">Majeur (4 critères)</span>
      <span class="text-slate-400 text-xs">vs</span>
      <span class="px-2 py-0.5 rounded bg-slate-100 text-slate-700 font-extrabold text-[9px] uppercase tracking-wider">Mineur (4 critères)</span>
    </div>
    <div class="grid grid-cols-2 gap-3">
      <div>
        <h4 class="font-extrabold text-xs text-amber-700 flex items-center gap-1"><i data-lucide="alert-triangle" class="w-3.5 h-3.5"></i> Majeur</h4>
        <ul class="text-[8px] text-slate-600 list-disc pl-3.5 space-y-1 mt-1">
          <li>Impacte un volume critique d'utilisateurs ou de sites</li>
          <li>Menace le respect d'une échéance réglementaire stricte</li>
          <li>Entraîne un risque de perte ou corruption de données</li>
          <li>Touche une fonctionnalité centrale (ex: imports CPAM)</li>
        </ul>
      </div>
      <div>
        <h4 class="font-extrabold text-xs text-slate-500 flex items-center gap-1"><i data-lucide="info" class="w-3.5 h-3.5"></i> Mineur</h4>
        <ul class="text-[8px] text-slate-600 list-disc pl-3.5 space-y-1 mt-1">
          <li>Impacte un seul utilisateur ou cas très isolé</li>
          <li>Il existe une solution de contournement simple</li>
          <li>Il s'agit d'un défaut cosmétique ou ergonomique simple</li>
          <li>Touche une fonctionnalité secondaire ou peu fréquente</li>
        </ul>
      </div>
    </div>
  </div>
</div>

# Reproductibilité d'un Bug
Qualifier si le comportement inattendu peut être recréé de manière systématique ou s'il dépend du contexte.

<!--
type: html
-->

<div class="grid grid-cols-1 md:grid-cols-2 gap-4 text-left w-full max-w-5xl mx-auto p-1" style="font-family: sans-serif;">
  <!-- Cas de Reproductibilité -->
  <div class="bg-white/90 border border-slate-200/60 p-4 rounded-xl shadow-xs">
    <h4 class="font-extrabold text-xs text-emerald-800 flex items-center gap-1.5 mb-2.5"><i data-lucide="refresh-cw" class="w-4 h-4 text-emerald-600"></i> Comment investiguer la reproductibilité ?</h4>
    <ul class="text-[9px] text-slate-600 flex flex-col gap-2.5">
      <li class="flex items-start gap-2">
        <i data-lucide="users" class="w-3.5 h-3.5 text-blue-900 shrink-0 mt-0.5"></i>
        <div>
          <strong class="text-slate-800">Autre compte / Autre instance :</strong>
          <span class="text-slate-500 block">Est-ce que le problème survient avec un autre profil utilisateur ou sur l'instance d'un autre client ?</span>
        </div>
      </li>
      <li class="flex items-start gap-2">
        <i data-lucide="server" class="w-3.5 h-3.5 text-blue-900 shrink-0 mt-0.5"></i>
        <div>
          <strong class="text-slate-800">Instance de pré-production :</strong>
          <span class="text-slate-500 block">Peut-on reproduire le bug dans l'environnement de test en créant un objet similaire du début ?</span>
        </div>
      </li>
      <li class="flex items-start gap-2">
        <i data-lucide="sliders" class="w-3.5 h-3.5 text-blue-900 shrink-0 mt-0.5"></i>
        <div>
          <strong class="text-slate-800">Problème de paramétrage :</strong>
          <span class="text-slate-500 block">Est-ce lié aux droits d'accès ou à une configuration spécifique de la structure ?</span>
        </div>
      </li>
    </ul>
  </div>

  <!-- Bug Aléatoire -->
  <div class="bg-white/90 border border-slate-200/60 p-4 rounded-xl shadow-xs flex flex-col justify-between">
    <div>
      <h4 class="font-extrabold text-xs text-amber-700 flex items-center gap-1.5 mb-2"><i data-lucide="help-circle" class="w-4 h-4"></i> Bug Aléatoire = Reproductibilité inconnue</h4>
      <p class="text-slate-500 text-[9px] leading-relaxed mb-3">Un bug qualifié d'« aléatoire » par l'utilisateur a toujours une cause logique cachée. Le rôle du support est d'identifier les variables contextuelles.</p>
    </div>
    <div class="bg-slate-50 border border-slate-100 p-3 rounded-lg space-y-2">
      <span class="text-[8px] font-extrabold text-slate-400 uppercase tracking-wider block">Pistes d'investigation :</span>
      <div class="grid grid-cols-2 gap-2 text-[8px] text-slate-600">
        <div class="flex items-center gap-1"><i data-lucide="clock" class="w-3 h-3 text-slate-400"></i> Expiration de session ?</div>
        <div class="flex items-center gap-1"><i data-lucide="mouse-pointer" class="w-3 h-3 text-slate-400"></i> Double-clic trop rapide ?</div>
        <div class="flex items-center gap-1"><i data-lucide="file" class="w-3 h-3 text-slate-400"></i> Données de fichiers mal formées ?</div>
        <div class="flex items-center gap-1"><i data-lucide="chrome" class="w-3 h-3 text-slate-400"></i> Navigateur obsolète ?</div>
      </div>
